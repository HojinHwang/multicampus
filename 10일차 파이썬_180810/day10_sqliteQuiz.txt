# SQL : insert into 테이블명 values('','','',,);


# 데이터 입력 - 두번째 방식 
# ?를 이용한 방식 
# sql변수= 'insert into 테이블명 values(?,?,?,?,?);'
# cursor.execute(sql변수,('','','', ,))


퀴즈 :
1) 테이블 생성 
   - 컬럼명 : 과목번호, 과목명, 교수명, 교수번호, 강의실코드, 강의실설명
                 : 영어로 지정 
   - 데이타 삽입 / 전체 데이타 조회 
2) 데이타 수정 
    강의실 설명값 변경하기 :  '공대301호' -> '공대505호'  
3) 과목명이 자바중급인 데이터를 삭제하여라 
4) 교수명이 '21'인 데이터만 출력해라
